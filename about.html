About This Game

Who's That Pokémon? is a fun fanmade mini project created by NwayOo to explore web development and have some coding fun. Inspired by the classic Pokémon anime segment, this game challenges players to guess the Pokémon silhouette before the timer runs out!

How It Works

  • A random Pokémon silhouette appears.
  • Type your guess before the timer ends.
  • Earn points for correct answers and build streaks.
  • Learn about Pokémon types, height, and weight.

Tech Stack

  • HTML, CSS, JavaScript
  • Data from PokéAPI
  • Audio & Sprites from original Pokémon series
